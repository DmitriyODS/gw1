Полигон

Яндекс.Диск — это сервис, который позволяет хранить файлы на удаленных серверах Яндекса и обмениваться ими. Данные доступны авторизованному пользователю с любого устройства, подключенного к интернету.
Вы можете оценить, как работает API Яндекс.Диска, отправляя «боевые» запросы из удобного интерфейса. Для каждой функции описаны параметры запроса и формат возвращаемых данных.
